¿Cómo funciona?
Tu colección se guarda en el archivo coleccion.json de tu repositorio. Como el repo es público, al abrir la página desde cualquier computadora verás la versión actual sin necesidad de token. El token solo se necesita en el dispositivo donde quieras editar y guardar cambios.

Cómo conseguir tu token (solo para editar):
  1. Abre github.com/settings/tokens/new
  2. Marca solo el scope public_repo
  3. Pon expiración (90 días recomendado)
  4. Click en Generate token
  5. Copia el token (empieza con ghp_) y pégalo abajo
